# 亲子徒步助手 🥾

一个帮助家长规划周末亲子徒步出行的 Web 应用。

## 功能特性

- 🗺️ **线路推荐**：根据城市和孩子年龄智能推荐合适的徒步线路
- 🌤️ **天气查询**：查看出行当天的天气情况
- 🚌 **交通指引**：公共交通到达方式建议
- 🎒 **装备建议**：根据天气和线路推荐装备和饮食

## 技术栈

- React 18 + Vite
- TailwindCSS
- 部署在 Vercel

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署到 Vercel

1. Fork 或克隆此仓库
2. 在 [Vercel](https://vercel.com) 导入项目
3. 自动部署完成
